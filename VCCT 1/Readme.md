# VCCT 1 
**Designed by Vcctus / Eiko @ VCCTUS**


The VCCT 1 is a high performance, bare metal processing board built around the Allwinner F1C200s. Designed for the "Mini Pico" form factor, it packs a 900MHz CPU and 64MB of internal DDR2 RAM into a tiny footprint. 

Unlike standard microcontrollers, the VCCT 1 is designed for massive compute tasks, advanced bare-metal C programming, and high-speed hardware control without the overhead of a full Linux operating system.

## Hardware Specifications
* **Processor:** Allwinner F1C200s (ARM926EJ-S @ 900MHz)
* **RAM:** 64MB DDR2 (Integrated in-package)
